# CrisisLink — Real Time Hotel Emergency Response System
Track: Rapid Crisis Response

# The Problem
Hotels have no structured real-time communication 
layer during emergencies. Guests panic, staff react 
slowly, emergency services arrive without context.
CrisisLink fixes this.

# What it does
- Guests report emergencies via one-tap interface
- Google Gemini AI classifies severity and generates 
  a 4-step response protocol instantly
- Staff see live alerts on a real-time dashboard
- Emergency services get a live incident briefing

# Response time: minutes → seconds

# Tech Stack
- Google Gemini AI — crisis classification
- Firebase Firestore — real-time database  
- Firebase Authentication — staff login
- Python FastAPI — AI backend
- JavaScript — frontend
- GitHub Pages — hosting

# How to run
1. Clone this repo
2. Open index.html in VS Code
3. Install Live Server extension
4. Right click index.html → Open with Live Server
